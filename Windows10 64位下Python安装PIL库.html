Windows10 64位下Python安装PIL库

发表于 2017-12-18 | 阅读次数

问题

PIL官方只有32位的Windows安装包，而我的操作系统是64位Windows 10，无法安装。

解决

  • 首先，在PIL非官方的64位库下载非官方64位pillow，注意下载符合自己Python的pillow。
  • 其次，DOS命令框下，注意此时的文件目录是你的Python里pip.exe所在的目录，像我的就是D:\python2.7\Scripts，然后输入pip install 你下载的pillow的路径，像我的就是pip install D:\360Downloads\Software\Pillow-4.1.0-cp27-cp27m-win_amd64.whl。
  • 最后在DOS命令框中输入Python，进入Python编辑，再输入from PIL import Image检验PIL是否配置成功。
